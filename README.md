**CulinAIry – AI Powered Chatbot for Personalized Food Recommendations**

**Overview**

CulinAIry is an AI-powered chatbot that provides personalized food recommendations based on user preferences such as diet, spice level, meal type, health goals, allergies, and dislikes.

It helps users make better food choices by balancing taste and nutrition through an interactive chat interface.

**Features**

- Chat-based interaction
- Personalized food suggestions
- Goal-based recommendations (weight loss / muscle gain)
- Allergy & dislike filtering
- Diet plan generation
- Nutrition tracking
- AI-based nutrition explanations
  
**Tech Stack**
- Backend: Python, Flask
- Database: SQLite, SQLAlchemy
- Frontend: HTML, CSS, JavaScript
- AI/NLP: RapidFuzz, Hugging Face API

**How It Works**

1.User inputs preferences

2.NLP extracts key details

3.Database is filtered

4.Recommendation logic selects best food

5.Result is displayed via chatbot

**Run Locally**

git clone https://github.com/your-username/culinairy.git

cd culinairy

pip install -r requirements.txt

python app.py

Open: http://localhost:5000
